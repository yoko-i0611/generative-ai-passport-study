# セキュリティとデータ保護について

## referenceContentの管理

公式テキストに基づく参照コンテンツ（`referenceContent`）は、以下の方法で保護されています：

### 1. Gitからの除外
- `data/reference-content.ts` ファイルは `.gitignore` に追加されています
- このファイルはGitリポジトリには含まれません

### 2. サーバーサイドでのみ提供
- `referenceContent`はサーバーサイドのAPIエンドポイント（`/api/reference-content`）経由でのみ提供されます
- クライアント側のコードには含まれていません
- ブラウザの開発者ツールからは直接内容を確認できません

### 3. 本番環境での設定

**重要**: 本番環境にデプロイする前に、以下を実行してください：

1. `data/reference-content.ts` ファイルをサーバーに手動で配置
2. または、環境変数や外部ストレージ（S3、データベースなど）から読み込むように設定

### 4. ローカル開発環境での設定

ローカルで開発する場合：

```bash
# data/reference-content.ts ファイルを作成
# このファイルはGitには含まれません
```

### 注意事項

- `data/reference-content.ts` ファイルには機密情報（公式テキストの内容）が含まれます
- このファイルをGitにコミットしないでください
- 公開リポジトリには絶対にプッシュしないでください
- チームメンバーとは安全な方法で共有してください（暗号化、専用チャンネルなど）
