# 404エラー対処方法

**作成日**: 2025-01-03  
**問題**: ブラウザで404エラーが発生している

---

## 🔍 問題の原因

404エラーが発生している原因：
- `.next`ディレクトリを削除したため、ビルドが必要
- 開発サーバーが再ビルド中

---

## ✅ 対処方法

### 1. 開発サーバーを再起動

```bash
cd /Users/imamurayoko/Developer/Cursor/application
# 既存のプロセスを停止
pkill -f "next dev"
# .nextディレクトリを削除
rm -rf .next
# 開発サーバーを起動
npm run dev
```

### 2. ビルドが完了するまで待つ

開発サーバーが起動すると、自動的にビルドが開始されます。通常、初回ビルドには30秒〜1分かかります。

### 3. ブラウザで確認

ビルドが完了したら、以下を確認：
- `http://localhost:3000/` にアクセス
- エラーが解消されているか確認

---

## 🔧 トラブルシューティング

### ビルドが完了しない場合

```bash
# 手動でビルドを実行
npm run build
# その後、開発サーバーを起動
npm run dev
```

### ポート3000が使用中の場合

```bash
# プロセスを停止
kill -9 $(lsof -ti:3000)
# 開発サーバーを再起動
npm run dev
```
