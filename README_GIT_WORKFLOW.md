## 🔄 推奨ワークフロー

### 1. 作業開始前

```bash
# 現在の状態を確認
git status

# ビルドが成功することを確認
npm run build

# ブラウザで正常に表示されることを確認
npm run dev
```

### 2. 各変更ごとに

```bash
# 変更を加える
# （ファイルを編集）

# Phaseチェックを実行
npm run phase-check <phase-number>

# 問題がなければコミット
git add <変更したファイル>
git commit -m "feat: <変更内容>"
```

### 3. エラーが発生した場合

```bash
# すぐに元に戻す
git checkout -- <変更したファイル>

# ビルドキャッシュをクリア
rm -rf .next

# 開発サーバーを再起動
npm run dev
```
