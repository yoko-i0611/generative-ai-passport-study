# 動作確認の手順

**作成日**: 2025-01-03  
**確認内容**: 第1章のページで`unit_quizzes_all.json`から確認テストが正しく読み込まれ、表示されることを確認

---

## 📋 確認項目

### 1. 開発サーバーの起動

```bash
cd /Users/imamurayoko/Developer/Cursor/application
npm run dev
```

ブラウザで `http://localhost:3000/` を開く

---

### 2. 第1章のページにアクセス

1. ホームページから「第1章 AI（人工知能）」をクリック
2. または直接 `http://localhost:3000/courses/chapter1` にアクセス

---

### 3. 確認すべき項目

#### ✅ 学習内容タブ

1. **ユニットカードの表示**
   - 4つのユニットカードが表示されるか
   - 各カードに画像、タイトル、要点が表示されるか

2. **テキストボタン**
   - 「テキスト」ボタンをクリックして、テキスト内容が表示されるか

3. **重要ポイントまとめボタン**
   - 「重要ポイントまとめ」ボタンをクリックして、重要ポイントが表示されるか

4. **確認問題ボタン**
   - 各ユニットに「確認問題」ボタンが表示されるか
   - ボタンに問題数（例: "(4問)"）が表示されるか

#### ✅ 確認問題モーダル

1. **モーダルの表示**
   - 「確認問題」ボタンをクリックして、モーダルが開くか
   - モーダルに問題が表示されるか

2. **問題の進行**
   - 問題を選択して「次の問題」ボタンで進めるか
   - 「前の問題」ボタンで戻れるか
   - 進捗バーが正しく更新されるか

3. **結果の表示**
   - すべての問題に回答して「結果を見る」をクリック
   - 正答率が表示されるか
   - 各問題の正解/不正解が表示されるか
   - 解説が表示されるか

#### ✅ 問題演習タブ

1. **問題の表示**
   - 「問題演習」タブをクリック
   - 全ユニットの問題がランダムに表示されるか
   - 問題を選択して進められるか

2. **結果の表示**
   - すべての問題に回答して結果を確認
   - 正答率が表示されるか

---

### 4. ブラウザのコンソール確認

1. ブラウザの開発者ツールを開く（F12または右クリック→検証）
2. コンソールタブを開く
3. エラーがないか確認
   - `unit_quizzes_all.json`の読み込みエラーがないか
   - その他のエラーがないか

---

### 5. 期待される動作

- ✅ `unit_quizzes_all.json`から確認テストが正しく読み込まれる
- ✅ 各ユニットに確認テストが統合される
- ✅ 確認問題ボタンをクリックすると、`unit_quizzes_all.json`のデータが表示される
- ✅ エラーが発生した場合は、既存の`quizQuestions`データが使用される（フォールバック）

---

## 🔍 トラブルシューティング

### `unit_quizzes_all.json`が読み込めない場合

1. `public/unit_quizzes_all.json`が存在するか確認
2. ファイルの形式が正しいか確認（JSON形式）
3. ブラウザのコンソールでエラーメッセージを確認

### 確認問題が表示されない場合

1. ブラウザのコンソールでエラーメッセージを確認
2. `unit_quizzes_all.json`の`sectionId`が`newChapter1Sections`の`id`と一致しているか確認
3. ネットワークタブで`unit_quizzes_all.json`の読み込みが成功しているか確認
